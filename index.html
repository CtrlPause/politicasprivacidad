Política de Privacidad para WhatsApp Business API Última actualización: 3 de febrero de 2026 Esta Política de Privacidad describe cómo nuestra aplicación maneja la información recopilada a través de los servicios de la API de WhatsApp Business. 1. Información que Recopilamos Nuestra aplicación recibe mensajes de texto y metadatos (como el número de teléfono y el ID de mensaje) enviados por los usuarios a través de WhatsApp. No recopilamos información personal adicional sin consentimiento explícito. 2. Uso de la Información La información recibida se utiliza exclusivamente para: Procesar consultas automáticas mediante nuestro AI Agent (Agente de IA). Gestionar la comunicación bidireccional entre el usuario y nuestra plataforma n8n. 3. Almacenamiento de Datos Los datos se procesan de forma transitoria. No vendemos ni compartimos historiales de chat con terceros. Los mensajes pueden ser almacenados temporalmente en nuestra instancia de Simple Memory (Memoria Simple) para mantener el contexto de la conversación. 4. Derechos del Usuario Los usuarios pueden solicitar la eliminación de su historial de chat enviando un mensaje directo a nuestra línea de atención. 5. Contacto Si tiene preguntas sobre esta política, puede contactarnos a través de nuestra plataforma de soporte. Siguientes pasos técnicos: Commit: Guarda los cambios en tu repositorio de GitHub.
